# RAG Template Starter - Planning Document

## Product Vision
Production-ready RAG starter kit for developers building AI applications with Azure OpenAI.

## Target Audience
- AI engineers implementing RAG systems
- Enterprise developers building AI features
- Startups adopting AI (GDPR-conscious EU market)

## Tech Stack

### Core
- **Runtime:** Node.js 20+ with TypeScript
- **LLM:** Azure OpenAI (GPT-4, GPT-4o)
- **Vector DB:** Azure AI Search (formerly Cognitive Search)
- **Embeddings:** Azure OpenAI text-embedding-3-large

### Infrastructure
- **Deployment:** Docker + Azure Container Apps
- **API:** Express.js or Fastify
- **Auth:** RBAC with Azure AD
- **Monitoring:** Azure Application Insights

### Development
- **Testing:** Vitest
- **Linting:** ESLint + Prettier
- **CI/CD:** GitHub Actions
- **Docs:** Markdown + TypeDoc
