# RAG Template Starter

Production-ready RAG (Retrieval-Augmented Generation) starter kit for Azure OpenAI + TypeScript.

## Overview

A complete, production-ready template for building RAG applications with:
- Azure OpenAI
- Azure AI Search
- TypeScript/Node.js
- Docker deployment
- GDPR-compliant architecture

## Quick Start

```bash
npm install
npm run dev
```

## Features

- [ ] Document ingestion pipeline
- [ ] Vector search with Azure AI Search
- [ ] Streaming responses
- [ ] Multi-modal support
- [ ] Docker deployment
- [ ] GDPR compliance utilities
